Задача 1. Линейный алгоритм "Перевод времени"

Алгоритм:

1. Получить входное значение "Часы".
2. Вычислить значение "Минуты" по формуле: "Минуты = Часы * 60".
3. Вывести полученное значение "Минуты".

Блок-схема:

[Начало] -> [Ввод: Часы] -> [Расчёт: Минуты = Часы * 60] -> [Вывод: Минуты] -> [Конец]

Задача 2. Алгоритм ветвления "Сравнение чисел"

Алгоритм:

1. Получить значения A и B.
2. Сравнить A и B.
3. Если A > B, тогда вывести сообщение: "Число A больше".
4. Иначе (в этом случае, так как числа различны, подразумевается B > A), вывести сообщение: "Число B больше".

Блок-схема:

[Начало] -> [Ввод: A, B] -> [Проверка: A > B?]
->(Да) [Вывод: "Число A больше"] -> [Конец]
->(Нет) [Вывод: "Число B больше"] -> [Конец]

Задача 3. Алгоритм ветвления "Допуск к экзамену"

Алгоритм:

1. Получить значение "Оценка".
2. Если "Оценка >= 3?", тогда вывести сообщение: "Допущен".
3. Иначе(если оценка меньше 3), вывести сообщение: "Не допущен".

Блок-схема:

[Начало] -> [Ввод: Оценка] -> [Проверка: Оценка >=3?]
->(Да) [Вывод: "Допущен"] -> [Конец]
->(Нет) [Вывод: "Не допущен"] -> [Конец]

Задача 4. Алгоритм с ветвлением "Скидка в магазине"

Алгоритм:

1. Получить значение "Цена" и "Количество".
2. Вычислить "ОбщаяСумма = Цена * Количество".
3. Если "ОбщаяСумма > 1000", тогда:
   -Вычислить "Скидка = ОбщаяСумма * 0.1" (или "ОбщаяСумма * 10 / 100").
   -Вычислить "Итог = ОбщаяСумма - Скидка".
4. Иначе (если сумма меньше или равна 1000):
   -"Итог = ОбщаяСумма"(без изменений).
5. Вывести значение "Итог".

Блок-схема:

[Начало] -> [Ввод: Цена, Количество] -> [Расчёт: Сумма = Цена * Количество] -> [Проверка: Сумма > 1000?]
->(Да) [Расчёт: Итог = Сумма - (Сумма * 0.1)] -> [Вывод: Итог] -> [Конец]
->(Нет) [Расчёт: Итог = Сумма] -> [Вывод: Итог] -> [Конец]

Задача 5. Алгоритм с вложенными ветвлениями (Квадратное уравнение)

Алгоритм:

1. Получить значение коэффицентов a, b,c.
2. Проверка: Если a равно 0, тогда:
   -Вывести сообщение: "Уравнение не является квадратным".
   -Перейти к концу алгоритма.
3. Иначе (если a не равно 0):
   -Вычислить дискриминант D = b² - 4 * a * c.
   -Вложенная проверка 1: Если D < 0, тогда:
    -Вывести сообщение: "Корней нет".
   -Вложенная проверка 2: Иначе, если D == 0, тогда:
    -Вычислить x = -b / (2 * a).
    -Вывести сообщение: "Один корень: [значние x]".
   -Вложенная проверка 3: Иначе (если D > 0):
    -Вычислить х1 = (-b + √D) / (2 * a).
    -Вычислить х2 = (-b - √D) / (2 * a).
    -Вывести сообщение: "Два корня: х1 = [значение], х2 = [значение]".

Блок-схема:

[Начало] -> [Ввод: a, b, c] -> [Проверка: a == 0?]
->(Да) [Вывод: "Уравнение не является квадратным"] -> [Конец]
->(Нет) [Расчёт: D = b*b - 4*a*c] -> [Проверка: D < 0?]
->->(Да) [Вывод: "Корней нет"] -> [Конец]
->->(Нет) [Проверка: D == 0?]
->->->(Да) [Расчёт: x = -b / (2*a)] -> [Вывод: "Один корень: ", x] -> [Конец]
->->->(Нет) [Расчёт: x1 = (-b + √D) / (2*a), x2 = (-b - √D) / (2*a)] -> [Вывод: "Два корня: ", x1, x2] -> [Конец]
